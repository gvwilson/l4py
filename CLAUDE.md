# Lean for Python Programmers

- This project is a one-day introduction to Lean for programmers who use Python
  on a daily basis, but have no previous experience with pure functional
  programming, advanced type systems, or theorem provers.

- Learners are primarily interested in using Lean to solve real-world problems,
  not in theory.

- Early lessons introduce ideas through lots of short examples; later lessons
  build small but useful applications.

- Learners are working on their own computers, not in a browser sandbox. They
  may be using VS Code, Emacs, Zed, or a plain text editor, so lessons must make
  minimal assumptions about tooling.

- Lessons must show common errors and explain how to debug and correct them.
